## Penjelasan code
Program ini terdiri dari tiga bagian utama yaitu HTML, CSS, dan JavaScript yang masing-masing punya peran berbeda tapi saling terhubung. Pada bagian HTML, kode digunakan untuk membuat struktur tampilan halaman. Di dalamnya terdapat dua section atau kartu yang merepresentasikan dua rangkaian lampu. Setiap rangkaian memiliki beberapa elemen <img> yang berfungsi sebagai lampu, dengan atribut src="off.png" sebagai kondisi awal (lampu mati). Selain itu, terdapat tombol-tombol (<button>) yang memiliki atribut onclick untuk memanggil fungsi JavaScript seperti toggleOne(), toggleAll(), dan toggleGroup(). Fungsi-fungsi ini nantinya akan mengontrol perubahan kondisi lampu. ID seperti r1 dan r2 digunakan untuk membedakan masing-masing rangkaian sehingga JavaScript bisa tahu lampu mana yang sedang dikontrol.

Masuk ke bagian CSS, kode ini bertugas mempercantik tampilan agar lebih menarik. Background dibuat dengan gradasi warna gelap (hitam ke abu-abu) supaya lampu terlihat lebih menonjol. Setiap rangkaian dibungkus dalam elemen dengan class .card yang diberi efek transparan, bayangan (shadow), dan sudut melengkung agar terlihat modern. Lampu ditata menggunakan flexbox melalui .lamp-container supaya tersusun rapi secara horizontal dan memiliki jarak antar lampu. Efek hover juga ditambahkan pada lampu dan tombol agar saat disentuh mouse terlihat ada animasi sedikit membesar, memberikan kesan interaktif.

Pada bagian JavaScript, di sinilah logika utama bekerja. Terdapat fungsi toggleLamp(lamp) yang bertugas mengubah gambar lampu dari mati ke nyala atau sebaliknya. Cara kerjanya adalah dengan mengecek apakah src dari gambar mengandung kata 'off.png'. Jika iya, maka akan diubah menjadi 'on.png' (lampu menyala), dan jika tidak maka dikembalikan ke 'off.png' (lampu mati). Fungsi toggleOne(containerId) digunakan untuk menyalakan atau mematikan lampu satu per satu secara bergantian. Variabel indexR1 dan indexR2 digunakan untuk menyimpan posisi lampu terakhir yang diubah, sehingga setiap tombol ditekan, lampu berikutnya yang akan berubah (seperti rangkaian seri). Fungsi toggleAll(containerId) akan mengambil semua lampu dalam satu rangkaian lalu menjalankan toggleLamp() pada masing-masing lampu, sehingga semua lampu berubah bersamaan (seperti rangkaian paralel). Terakhir, fungsi toggleGroup() khusus untuk rangkaian kedua yang memiliki empat lampu, di mana lampu dibagi menjadi dua kelompok yaitu (a,b) dan (c,d), lalu masing-masing kelompok dinyalakan atau dimatikan secara bersamaan.
